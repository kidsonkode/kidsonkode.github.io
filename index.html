The kids will learn basic concepts about Website design and be able to create their own website from scratch.

The class will start with basic understanding HTML (HyperText Markup Language) which represents the framework of a web site. Once the framework is layed out, we will turn our attention to styling and cosmetics by learning about CSS (Cascading Style Sheets) and JavaScript.
